# Code Of Conduct

This project aims to be a welcoming, practical place to collaborate.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, especially across time zones and language backgrounds.
- Keep discussion focused on the code, documentation, and user impact.
- Give maintainers enough context to reproduce problems.
- Respect private security reporting for vulnerabilities.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or conduct.
- Publishing another person's private information.
- Posting credentials, private data, exploit details, or organization-only
  material in public project spaces.
- Repeated off-topic or bad-faith disruption.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues or pull requests;
or restrict participation when needed to protect the project community.

To report a conduct concern, contact the maintainers privately. Reports should
include links, screenshots, or other context when available.
